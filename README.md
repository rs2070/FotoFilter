# FotoFilter

![FotoFilter Screenshot](https://i.postimg.cc/gk3LFTjN/ezgif-5-b8a16a7622.gif)

FotoFilter assists in effectively organizing, filtering, and removing undesirable images. This tool employs smart algorithms to identify duplicate photos, detect faces, and evaluate blurriness. The application is lightweight, portable, and cross-platform compatible when it is deployed using Docker.

## **About**
This repository holds the source code for FotoFilter. To run the application locally:

```bash
git clone https://github.com/rs2070/FotoFilter.git
cd FotoFilter
pip install -r requirements.txt
python app.py
```
